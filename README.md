A Chrome extension that brings AI directly into your workflow.

**Select text** anywhere on the web and get instant explanations.

Or use **Boom** to capture any part of your screen and ask questions about diagrams, charts, code, formulas, UI layouts, PDFs, images, and anything visual.

No tab switching. No copy-pasting. No breaking focus.

## Selection

Highlight any text on any webpage.

A floating AI prompt appears directly above your selection.

Ask:

* Explain simply
* Summarize
* Give examples
* Explain like I'm 12
* Translate
* What does this mean?

SnB keeps the highlighted context visible while generating the answer.

Works great for:

* Research papers
* Documentation
* Technical blogs
* Online courses
* Academic PDFs
* Stack Overflow threads

## Boom

Sometimes text is not enough.

Use **Boom** to capture any region of your screen.

SnB can analyze:

* Charts
* Graphs
* Tables
* UI layouts
* Code screenshots
* Mathematical formulas
* Academic figures
* Diagrams
* Error messages

Ask questions naturally:

> "Explain this graph."

> "What's wrong with this code?"

> "Summarize this diagram."

> "Explain this formula."

> "What is this architecture showing?"

## Install

```bash
git clone https://github.com/yourusername/SnB.git
cd SnB
```

Load the extension in Chrome and start selecting text or using Boom.

## Privacy

SnB only processes content you explicitly select or capture.

* No background page scraping
* No automatic data collection
* No hidden tracking
* User-controlled API keys
* Explicit user actions only

Your workflow stays yours.

## Why SnB?

Most AI tools require you to leave what you're doing.

SnB brings AI directly into the page you're already reading.

Highlight.

Boom.

Understand.
